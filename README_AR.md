# مستكشف حرب النجوم (Star Wars Explorer) 🌌

مشروع تفاعلي حديث لاستكشاف مجرة حرب النجوم، تم بناؤه باستخدام **React** و **TypeScript**.

## ✨ المميزات الرئيسية

- **🎬 استعراض الأفلام**: واجهة سينمائية تعرض جميع أفلام السلسلة مع بطاقات قابلة للسحب وتأثيرات بصرية جذابة.
- **👥 دليل الشخصيات**: قائمة شاملة لشخصيات المجرة مع نظام بحث فوري وذكي.
- **🔍 تفاصيل عميقة**: صفحات مخصصة لكل شخصية تعرض بياناتها التقنية (مثل الطول، الوزن، الموطن الأصلي) المستخرجة من أرشيفات SWAPI.
- **🌗 نظام الألوان (Theme)**: مبدل سمات متكامل يسمح بالتبديل بين الوضع المظلم والوضع الفاتح مع الحفاظ على الحالة عبر التطبيق.
- **🚀 أداء محسن**: استخدام تقنيات الميموزيشن (`useMemo`, `useCallback`, `React.memo`) لضمان سرعة فائقة وتجربة مستخدم سلسة.
- **🎮 عناصر ثلاثية الأبعاد**: دمج شعار حرب النجوم بتقنية 3D التفاعلية باستخدام React Three Fiber.

## 🛠 التقنيات المستخدمة

- **React 18** & **TypeScript**
- **Vite** (بيئة تطوير سريعة جداً)
- **Tailwind CSS** (للتصاميم المتقدمة والمستجيبة)
- **React Router** (لإدارة التنقل بين الصفحات)
- **React Three Fiber & Drei** (للرسوميات ثلاثية الأبعاد)
- **Context API** (لإدارة الحالة العالمية للقوالب)

## 🚀 التشغيل السريع

1. قم بتثبيت المكتبات اللازمة:
   ```bash
   npm install
   ```

2. ابدأ تشغيل خادم التطوير:
   ```bash
   npm run dev
   ```

3. افتح المتصفح على: `http://localhost:5173`
